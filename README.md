# Lexington Global Systems – Enterprise Data Entry Platform

A full-stack enterprise workflow platform designed for large-scale data entry, verification, and micro-task management operations.

## Live Demo

https://lexington-data-entry-platform.onrender.com

## Features

* Role-based dashboards for Workers, Clients, and Admins
* Authentication and access control
* Task and workflow management
* REST API backend services
* PostgreSQL database integration
* Responsive enterprise UI
* Earnings and activity tracking

## Tech Stack

* React
* TypeScript
* Node.js
* Express.js
* PostgreSQL
* Supabase
* Vite

## Architecture

The platform follows modular full-stack architecture principles with REST API communication between frontend and backend services.

## Deployment

* Frontend + Backend: Render
* Database: Supabase

## Local Setup

```bash
npm install
npm run dev
```
